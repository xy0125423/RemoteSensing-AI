### 2.3 植被指数（Vegetation Indices）

```
NDVI（核心）+ NDWI + EVI（进阶）
├── 为什么学：NDVI 是遥感中最基础、最广泛使用的植被指数。理解它 = 理解光谱分析
├── 在项目中的位置：Phase 2，NDVI 计算引擎
├── 知识点：
│   ├── NDVI = (NIR - Red) / (NIR + Red)
│   ├── 为什么 NIR 反射率高 → 植被健康
│   ├── 为什么 Red 吸收强 → 光合作用
│   ├── NDVI 值范围 [-1, 1]
│   │   ├── >0.5：茂密植被
│   │   ├── 0.2~0.5：稀疏植被
│   │   ├── 0~0.2：裸土
│   │   └── <0：水体/云/雪
│   └── NDVI 的局限性（饱和、大气影响）
└── 与考研的关系：遥感原理 → 植被光谱特征 → 植被指数章节（核心考点）
```

### 2.5 傅里叶分析（FFT）

```
傅里叶分析 ←── 与考研 892 最紧密的知识点
├── 为什么学：
│   1. NDVI 时序中的周期成分（年周期、生长季周期）需要 FFT 提取
│   2. 你正在备考 892 信号与线性系统，FFT 是核心考点
│   3. 遥感 + 信号处理 = 你的独特竞争优势
├── 在项目中的位置：Phase 4，FFT 频谱分析
├── 知识点：
│   ├── DFT / FFT 原理（蝶形运算不需要手写，但需要理解）
│   ├── 采样定理（奈奎斯特频率）
│   │   └── Sentinel-2 重访周期 5 天 → 最高可检测频率 = 1/(2×5) = 0.1 天⁻¹
│   │   └── 即：能检测到的最短周期 = 10 天
│   ├── 频率分辨率 = 1/(N×Δt)
│   │   └── N=365 天, Δt=1 天 → 分辨率 = 1/365 ≈ 0.0027 天⁻¹
│   ├── 幅度谱 vs 功率谱
│   ├── 频谱泄漏与加窗
│   └── 从频谱到周期：周期(天) = 1/频率(天⁻¹)
└── 与考研的关系：892 信号与系统 → 傅里叶变换（第 3~5 章，核心中的核心）
```

## 与考研 892 信号与线性系统的知识关联

| 892 知识点 | 项目对应 | 关联方式 |
|------------|----------|----------|
| 信号的时域描述 | NDVI 时间序列 = 离散时间信号 x[n] | 直接映射 |
| 采样定理 | Sentinel-2 重访周期 5 天 → fs=0.2 天⁻¹ → fmax=0.1 天⁻¹ | 理解数据限制 |
| 离散傅里叶变换（DFT） | `scipy.fft.rfft()` 对 NDVI 序列做频谱分析 | 实践 DFT |
| 频谱/幅度谱/功率谱 | 频谱图 → 识别植被变化周期 | 频谱解读 |
| 频域分辨率 | Δf = 1/(N×Δt)，影响能分辨的周期精度 | 参数选择 |
| 线性系统 | NDVI 时间序列的趋势+季节分解 | 系统思维 |
| 滤波器 | 滑动平均 ≈ 低通滤波器 → 去除高频噪声 | 滤波器应用 |

**关键策略**：做项目的过程中，每遇到一个 FFT 相关的操作，都去翻 892 课本的对应章节。这样项目做完了，考研复习也同步推进了。
