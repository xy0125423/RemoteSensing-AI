## Phase 1：GeoTIFF 驯服战

**时间**：2026-07-03 ~ 2026-07-20（约 18 天，54 小时）

**目标**：打通"GEE → 本地 GeoTIFF → rasterio 读取 → matplotlib 显示"全链路。

### 子目标

1. **GEE 导出**：使用 `ee.batch.Export.image.toDrive()` 导出单景 Sentinel-2 影像到 Google Drive
2. **本地下载**：从 Google Drive 下载 GeoTIFF 到 `data/raw/`
3. **rasterio 读取**：读取 GeoTIFF 的元数据（CRS、Transform、Bounds、波段数）
4. **波段提取**：提取 B4（Red）和 B8（NIR）为独立 numpy 数组
5. **真彩色显示**：使用 matplotlib 显示 B4-B3-B2 真彩色合成图
6. **批量导出**：学会批量导出多景影像

### Phase 1 完成标志

```
✅ 能导出至少 1 景 Sentinel-2 GeoTIFF 到本地
✅ 能用 rasterio 读取并打印元数据
✅ 能用 matplotlib 显示真彩色影像
✅ 能提取单个波段为 numpy 数组
```

### 依赖

- Phase 0（已完成）

### 预计时间

约 54 小时（18 天 × 3 小时/天）

## Phase 2：NDVI 计算引擎

**时间**：2026-07-21 ~ 2026-08-15（约 26 天，78 小时）

**目标**：建立 NDVI 计算流水线，能手写 NDVI 公式而不是调黑盒函数。

### 子目标

1. **理解 NDVI 原理**：NDVI = (NIR - Red) / (NIR + Red)，理解为什么这个公式能反映植被
2. **手写 NDVI 计算**：用 numpy 对单景影像计算 NDVI（不依赖库函数）
3. **NDVI 可视化**：用 matplotlib 显示 NDVI 伪彩色图（绿=高植被，棕=低植被）
4. **NDVI 统计**：计算 NDVI 均值、标准差、直方图
5. **多景批量 NDVI**：对 Phase 0 筛选的 28 景影像，逐景计算 NDVI 均值
6. **NDVI 数据持久化**：将每景的 (日期, NDVI均值) 保存为 CSV

### Phase 2 完成标志

```
✅ 能手写 NDVI = (NIR - Red) / (NIR + Red)
✅ 能生成 NDVI 伪彩色图
✅ 能批量计算 28 景影像的 NDVI 均值
✅ 输出 data/processed/ndvi_series.csv
```

### 依赖

- Phase 1（需要本地 GeoTIFF 或 GEE 直接计算）

### 预计时间

约 78 小时（26 天 × 3 小时/天）

## Phase 3：时间序列分析

**时间**：2026-08-16 ~ 2026-09-15（约 31 天，93 小时）

**目标**：从离散的 NDVI 数据点构建完整的时间序列，并进行分析。

### 子目标

1. **时间序列概念**：理解什么是时间序列，为什么 NDVI 序列是"非均匀采样"
2. **数据清洗**：处理缺失值、异常值（云漏检导致 NDVI 异常低）
3. **插值**：对非均匀采样的 NDVI 序列进行插值，得到均匀时间步长
4. **滑动平均**：对 NDVI 序列做滑动窗口平滑，去除噪声
5. **趋势分解**：分离 NDVI 序列的趋势项和季节项
6. **物候特征提取**：提取 NDVI 最大值日期（峰值）、生长季起始/结束日期
7. **多区域对比**（可选）：对比不同土地利用类型的 NDVI 时序

### Phase 3 完成标志

```
✅ 能从 ndvi_series.csv 构建时间序列
✅ 能对非均匀采样数据插值
✅ 能绘制平滑后的 NDVI 时序折线图
✅ 能提取至少 3 个物候指标（峰值日期、生长季长度、NDVI 振幅）
```

### 依赖

- Phase 2（需要 NDVI 序列数据）

### 预计时间

约 93 小时（31 天 × 3 小时/天）

## Phase 4：傅里叶频谱分析

**时间**：2026-09-16 ~ 2026-10-20（约 35 天，105 小时）

**目标**：使用傅里叶变换分析 NDVI 时间序列的周期成分，将信号与系统理论应用于遥感实践。

### 子目标

1. **FFT 基础复习**：结合 892 考研内容，复习 DFT、FFT、频谱、采样定理
2. **NDVI 序列 FFT**：对插值后的均匀 NDVI 序列执行 `scipy.fft.fft()`
3. **频谱图绘制**：绘制幅度谱和功率谱，横轴为频率/周期
4. **主周期提取**：从频谱中提取振幅最大的频率分量，换算为天数
5. **频谱解读**：解释主周期的物理含义（如：年周期=365天、半年周期=182天、生长季周期）
6. **多区域频谱对比**（可选）：不同植被类型的频谱差异

### Phase 4 完成标志

```
✅ 能对 NDVI 序列执行 FFT 并绘制频谱图
✅ 能从频谱中提取 Top 3 主要周期成分
✅ 能用人话解释频谱图的含义（而不只是"调了 scipy.fft"）
✅ 频谱解读与 892 考研知识点形成闭环
```

### 依赖

- Phase 3（需要均匀采样的 NDVI 时间序列）

### 预计时间

约 105 小时（35 天 × 3 小时/天）
